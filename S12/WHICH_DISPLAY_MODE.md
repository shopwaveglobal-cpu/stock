# 로그 표시 모드 선택

## 옵션 1: 자동 로그 창 열기 ⭐⭐⭐

### 동작
```
08:00: 모니터링 시작
       + 로그 창 자동으로 열림
       → 화면에 실시간 로그 표시
20:00: 모니터링 종료
       + 로그 창 자동 닫힘

다음날 08:00: 다시 자동 시작!
```

### 장점
- ✅ 항상 로그 확인 가능
- ✅ 실시간 감시 느낌
- ✅ 문제 즉시 발견

### 단점
- ⚠️ 로그 창 항상 켜져 있음
- ⚠️ 전력 약간 증가
- ⚠️ 화면 공간 차지

---

## 옵션 2: 필요할 때만 확인 (현재) ⭐⭐⭐⭐⭐

### 동작
```
08:00: 모니터링 시작 (백그라운드)
사용자: watch_logs_lite.bat 실행 (선택사항)
→ 1분 후 자동 닫힘

20:00: 모니터링 종료
```

### 장점
- ✅ 전력 절약
- ✅ 화면 깔끔
- ✅ Telegram 알림만으로 충분
- ✅ 효율적

### 단점
- ⚠️ 수동으로 창 열어야 함

---

## 추천

**옵션 2 (현재 방식)가 실용적!**

- 무인 운영에는 Telegram 알림이 최선
- 필요할 때만 로그 확인
- 전력 절약 + 화면 깔끔
