## 📊 포맷 정보

### 가격 포맷
- 천 단위 콤마: `{int(price):,}원`
- 예: `77,800원`, `1,075,000원`

### 이격도 포맷
- 실시간 알람: `{distance_pct:+.2f}%` (소수점 2자리)
- 일일 리포트: `{dist:.1f}%` 또는 `{dist:+.1f}%` (소수점 1자리)
- 예: `+3.05%`, `-0.26%`, `+96.0%`

### 시간 포맷
- 실시간 알람: `HH:MM:SS` (예: `14:25:30`)
- 일일 리포트: `YYYY-MM-DD HH:MM` (예: `2025-11-02 20:10`)

### 구분선
- 실시간 알람: `───────────` (11개)
- 일일 리포트: `───────────────` (15개)

## 📝 참고사항

1. **저가 정보**: 현재 텔레그램 실시간 알람에는 저가 정보가 없음 (슬랙에는 있음)
2. **매수선 인접 세부 타입**: 1%, 3%, 5% 인접 이모지가 코드에 없음 (추가 필요)
3. **일일 리포트 순서**: 1차 매수 → 2차 매수 → 3차 매수 → 매수 완료 → 매도선 접근
4. **정렬 기준**:
   - 매수 접근 중: 이격도 낮은 순
   - 매수 완료: 수익률 높은 순
   - 매도선 접근: 이격도 낮은 순
