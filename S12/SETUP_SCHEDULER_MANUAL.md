# Windows 작업 스케줄러 수동 설정 가이드

## 방법 1: 스크립트로 자동 설정 (권장)

### 1단계: PowerShell 관리자 권한으로 실행
- Win + X → "Windows PowerShell (관리자)" 선택
- 또는 시작 메뉴에서 PowerShell 우클릭 → "관리자 권한으로 실행"

### 2단계: 스크립트 실행
```powershell
cd C:\Users\log\Desktop\Code\S12
.\setup_windows_scheduler.ps1
```

## 방법 2: 수동 설정

### 1단계: 작업 스케줄러 열기
- **Win + R** → `taskschd.msc` 입력 → Enter

### 2단계: 작업 1 생성 (Daily Turnover + Signal)

1. **"작업 만들기"** 클릭

2. **일반 탭**
   - 이름: `S2_Daily_Trading_Signal`
   - 설명: S12 매일 거래대금 + 시그널 분석
   - **"사용자가 로그온할 때만 실행"** 체크
   - **"가장 높은 수준의 권한으로 실행"** 체크

3. **트리거 탭**
   - **"새로 만들기"** 클릭
   - 작업 시작: **일정에 따라**
   - 설정: **매일**
   - 시작: **2025-11-02 20:10**
   - 반복: **1일**
   - 사용 안 함: **체크 해제**
   - **확인** 클릭

4. **동작 탭**
   - **"새로 만들기"** 클릭
   - 동작: **프로그램 시작**
   - 프로그램/스크립트: `C:\Users\log\Desktop\Code\S12\RUN_DAILY_SYSTEM.bat`
   - **확인** 클릭

5. **조건 탭**
   - ❌ **"컴퓨터의 전원이 켜져 있을 때만 작업 시작"** 해제
   - ❌ **"AC 전원일 때만 시작"** 해제

6. **설정 탭**
   - ✅ **"작업이 실패하면 다시 시작"** 체크
   - 다시 시작 간격: **1분**
   - 다시 시작 시도 횟수: **3번**

7. **확인** 클릭

### 3단계: 작업 2 생성 (Real-Time Monitor)

1. **"작업 만들기"** 클릭

2. **일반 탭**
   - 이름: `S2_Realtime_Monitor`
   - 설명: S12 실시간 주식 모니터링
   - **"사용자가 로그온할 때만 실행"** 체크
   - **"가장 높은 수준의 권한으로 실행"** 체크

3. **트리거 탭**
   - **"새로 만들기"** 클릭
   - 작업 시작: **일정에 따라**
   - 설정: **주간**
   - 요일: **월, 화, 수, 목, 금** 체크
   - 시작: **2025-11-03 08:00**
   - 반복: **매 1주**
   - 사용 안 함: **체크 해제**
   - **확인** 클릭

4. **동작 탭**
   - **"새로 만들기"** 클릭
   - 동작: **프로그램 시작**
   - 프로그램/스크립트: `C:\Users\log\Desktop\Code\S12\run_real_time_monitor.bat`
   - **확인** 클릭

5. **조건 탭**
   - ❌ **"컴퓨터의 전원이 켜져 있을 때만 작업 시작"** 해제
   - ❌ **"AC 전원일 때만 시작"** 해제

6. **설정 탭**
   - ✅ **"작업이 실패하면 다시 시작"** 체크
   - 다시 시작 간격: **1분**
   - 다시 시작 시도 횟수: **3번**

7. **확인** 클릭

## 테스트

### 수동 실행 테스트

1. 작업 스케줄러에서 **"S2_Daily_Trading_Signal"** 선택
2. 우클릭 → **"실행"**
3. 로그 확인:
   ```
   C:\Users\log\Desktop\Code\S12\logs\s12_daily_YYYYMMDD.log
   ```
4. 텔레그램 메시지 도착 확인

## 7단계: 24/7 실행 환경 설정

### Windows 절전 모드 비활성화

1. **설정** → **시스템** → **전원 및 절전**
2. 다음 설정 변경:
   - 화면: **안 함**
   - 절전 모드: **안 함**

### Windows 업데이트 자동 재시작 방지

1. **설정** → **업데이트 및 보안** → **Windows Update**
2. **"고급 옵션"** 클릭
3. **"활성 시간"** 설정:
   - 시작 시간: **00:00**
   - 종료 시간: **23:59**
4. **"다시 시작하는 옵션을 확인하세요"** → **"활성 시간 동안 다시 시작하지 않음"** 체크

### 네트워크 연결 유지

1. **Win + X** → **"네트워크 연결"**
2. 네트워크 어댑터 우클릭 → **"속성"**
3. **"구성"** 클릭
4. **"전원 관리"** 탭
5. ❌ **"전원 절약을 위해 컴퓨터가 이 장치를 끌 수 있음"** 해제
6. **확인** 클릭
