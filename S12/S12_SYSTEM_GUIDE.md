# S12 트레이딩 시스템 가이드

## 🎯 시스템 개요
- **프로젝트 목적**: 20일 이평선 엔벨로프 기반 매매 시그널 시스템
- **핵심 기능**: 실시간 모니터링, 매수/매도 시그널 생성, 텔레그램 알림
- **데이터 소스**: Kiwoom REST API

## 🔄 주요 작동 로직

### 데이터 흐름
```
Daily_Turnover_Tracker.py → turnover_universe.xlsx → Trading_Signal_System.py → trading_signals.xlsx → Real_Time_Monitor.py
```

### 매수 전략 (3단계 분할 매수)
- **1차 매수선**: 엔벨로프 + 1틱
- **2차 매수선**: 1차 매수선 - 10% + 1틱  
- **3차 매수선**: 2차 매수선 - 10% + 1틱

### 매도 전략
- **+3%**: 1차 매도선 도달 시 재터치 매도
- **+5%**: 2차 매도선 도달 시 재터치 매도
- **+7%**: 3차 매도선 도달 시 즉시 전량 매도

## 📁 파일 구조 및 역할

### 핵심 파일 (수정 시 주의)
- `Daily_Turnover_Tracker.py`: 거래대금 5000억+ 종목 수집
- `Trading_Signal_System.py`: 매매 시그널 생성 및 텔레그램 리포트
- `Real_Time_Monitor.py`: 실시간 모니터링
- `telegram_notifier.py`: 텔레그램 알림 전송
- `run_trading_signal.bat`: Windows Task Scheduler 실행 파일

### 데이터 파일 (output 폴더에만 저장)
- `output/turnover_universe.xlsx`: 거래대금 5000억+ 종목 목록 (축적식)
- `output/trading_signals.xlsx`: 현재 분석 결과 (Summary + History 탭)
- `output/market_cap_universe.xlsx`: 시가총액 필터링 결과

### 폴더 구조
- `test/`: 모든 테스트, 디버그, 체크 파일들
- `s1/`: S1 시스템 관련 파일들 (별도 시스템)
- `old/`: 가이드 문서, 로그 파일, 레거시 파일들

## 🔧 기술적 세부사항

### API 설정
- **Base URL**: `https://api.kiwoom.com`
- **인증**: Bearer Token 방식
- **차트 데이터**: KRX+NXT 통합 기준 (`_AL` 접미사)

### 실시간 모니터링 시스템
- **모니터링 시간**: 08:00-20:00
- **동적 간격**: 1% 이내(1분), 3% 이내(3분), 10% 이내(10분), 10% 이상(30분)
- **저가 기준 터치 감지**: 저가가 매수선에 도달 시 체결 알림

### 알림 로직 (진행 상황 반영)
- **접근 알림**: 더 가까워질 때만 새로운 알림 발생
  - 1% 이격 → 🔴 "1% 인접" 알림
  - 3% 이격 → 🟠 "3% 인접" 알림 (1% 알림이 없었던 경우만)
  - 5% 이격 → 🟡 "5% 인접" 알림 (1%, 3% 알림이 없었던 경우만)
- **체결 알림**: 저가가 매수선 터치 시 항상 발생 🎯
- **중복 방지**: 같은 레벨 알림은 하루 1회만 전송

### 텔레그램 알림 시스템
- **일일 리포트**: 매일 20:10 자동 전송
- **정렬 순서**: 이격도 낮은 순 (가장 중요한 종목부터)
- **수신자**: 4명 (me, yoonjoo, minjeong, jumeoni)

### Excel 포맷팅
- **이격도**: `0.00"%"` 형식 (이미 % 값이므로 % 기호만 추가)
- **금액**: `#,##0` 형식 (천 단위 콤마)
- **날짜**: `YYYY-MM-DD` 형식

## 🐛 문제 해결 히스토리

### 해결된 문제들
1. **중복 제거**: 기존 데이터 유지 로직 제거로 중복 해결
2. **이격도 포맷팅**: Excel에서 올바른 % 표시 수정
3. **텔레그램 정렬**: 이격도 낮은 순으로 정렬 개선
4. **파일 구조**: 테스트/S1 파일 분리로 정리
5. **저가 데이터 오류**: API 응답 키 매핑 수정 (거래량→저가 오류 해결)
6. **알림 우선순위**: 더 가까운 조건부터 체크하도록 로직 수정
7. **알림 진행 로직**: 더 가까워질 때만 새로운 알림 발생하도록 개선
8. **Excel 양식 고정 및 시트 이름 자동 감지** (2025-11-09):
   - pandas ExcelWriter 대신 openpyxl 직접 사용으로 전환
   - Unnamed 컬럼 완전 제거 (G1: None으로 유지)
   - 날짜 포맷: yyyy-mm-dd (시간 제거)
   - 테두리: 값이 있는 셀만 적용
   - H1 셀: "최종 업데이트: YYYY-MM-DD" (테두리 없음, 왼쪽 정렬)
   - **시트 이름 자동 감지**: 'universe' 시트가 없으면 첫 번째 시트 자동 읽기
   - **데이터 복원**: 11/6~11/7 손실 데이터를 로그에서 복원 완료
