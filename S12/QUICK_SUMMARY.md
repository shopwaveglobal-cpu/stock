# 빠른 요약

## 로그 모니터링 도구

### 1. watch_logs_lite.bat ⭐ 추천
```
사용법: watch_logs_lite.bat
- 한 번만 표시 후 종료
- 전력 소비 거의 없음
- 확인 필요시에만 실행
```

### 2. watch_logs.ps1
```
사용법: .\watch_logs.ps1 -Type today
- 실시간 업데이트
- 전력 약간 소비 (~150원/월)
- 디버깅 시에만 사용
```

## 전력 소비 비교

| 방법 | 월 추가 전력비 | 추천 상황 |
|------|--------------|----------|
| 무모니터링 | 0원 | 평상시 |
| Lite 모드 | 30원 | 하루 1회 확인 |
| 실시간 | 150원 | 디버깅 시 |

## 결론

**추천:**
1. 평상시: 모니터링 안 함 (Telegram 알림만)
2. 확인 필요: `watch_logs_lite.bat` 실행
3. 디버깅: `watch_logs.ps1` 실행

**예상 추가 전력비: ~30원/월**
