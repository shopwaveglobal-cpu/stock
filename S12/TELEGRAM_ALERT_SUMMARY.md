## 📱 텔레그램 알람 시스템 구조

### 1. **일일 트레이딩 리포트** (Daily Trading Report)

#### 실행 시점
- **Trading_Signal_System.py** 실행 후 (매일 20:10 자동 실행)
- 장마감 후 일괄 분석 결과 리포트

#### 기준 값 및 조건
- **기준 파일**: `output/trading_signals.xlsx` (Summary 시트)
- **알람 대상**: `알람상태`가 다음 중 하나인 종목
  - `READY_BUY1`: 1차 매수선까지 10% 이내
  - `READY_BUY2`: 2차 매수선까지 10% 이내
  - `READY_BUY3`: 3차 매수선까지 10% 이내
  - `READY_SELL1`: +3% 매도선까지 10% 이내
  - `READY_SELL2`: +5% 매도선까지 10% 이내
  - `READY_SELL3`: +7% 매도선까지 10% 이내
  - 매수 완료 종목 (`BOUGHT_1`, `BOUGHT_2`, `BOUGHT_3`)

#### 전송 내용

**1. 1차 매수 접근 중** (READY_BUY1)
- 종목명
- 현재가
- 1차 매수선 (익일 기준)
- 1차 매수선 이격도 (%)
- **정렬**: 이격도 낮은 순

**2. 매수 완료 종목** (BOUGHT_1, BOUGHT_2, BOUGHT_3)
- 종목명
- 현재가
- 평균 매수가
- 이격도 (현재가 대비 평균 매수가)
- **정렬**: 수익률 높은 순

**3. 매도선 접근** (READY_SELL1, READY_SELL2, READY_SELL3)
- 종목명
- 현재가
- 목표가 (매도선)
- 이격도 (%)
- **정렬**: 이격도 낮은 순

#### 메시지 포맷 예시
```
📊 일일 트레이딩 리포트
🕐 2025-11-02 20:10
───────────────

🟡 1차 매수 접근 중 (2개)
  • 삼성전자
    현재가: 107,700원
    매수가: 75,500원
    이격도: -29.9%

🔴 매수 완료 종목 (5개)
  • SK하이닉스
    현재가: 560,000원
    평균가: 138,000원
    이격도: +305.8%

🟢 매도선 접근 (1개)
  • NAVER
    현재가: 270,500원
    목표가: 225,570원
    이격도: +19.9%
```

#### 전송 대상
- **수신자**: 모든 사람 (`recipients=["all"]`)
- **전송 시점**: Trading Signal System 분석 완료 후

### 2. **실시간 알람** (Realtime Alert)

#### 실행 시점
- **Real_Time_Monitor.py** 실행 중
- 실시간 가격 모니터링 중 조건 충족 시 즉시 전송

#### 기준 값 및 조건

**매수선 알람** (저가 기준 이격도)

| 매수 상태 | 조건 | 알람 타입 | 이격도 기준 |
|----------|------|----------|------------|
| **NONE** | 저가 ≤ 1차 매수선 | `1차 매수 체결!` | 저가 기준 이격도 ≤ 0% |
| **NONE** | 저가 기준 1% 이내 | `1차 매수선 1% 인접` | 0% < 이격도 ≤ 1% |
| **NONE** | 저가 기준 3% 이내 | `1차 매수선 3% 인접` | 1% < 이격도 ≤ 3% |
| **NONE** | 저가 기준 5% 이내 | `1차 매수선 5% 인접` | 3% < 이격도 ≤ 5% |
| **BOUGHT_1** | 저가 ≤ 2차 매수선 | `2차 매수 체결!` | 저가 기준 이격도 ≤ 0% |
| **BOUGHT_1** | 저가 기준 1% 이내 | `2차 매수선 1% 인접` | 0% < 이격도 ≤ 1% |
| **BOUGHT_1** | 저가 기준 3% 이내 | `2차 매수선 3% 인접` | 1% < 이격도 ≤ 3% |
| **BOUGHT_1** | 저가 기준 5% 이내 | `2차 매수선 5% 인접` | 3% < 이격도 ≤ 5% |
| **BOUGHT_2** | 저가 ≤ 3차 매수선 | `3차 매수 체결!` | 저가 기준 이격도 ≤ 0% |
| **BOUGHT_2** | 저가 기준 1% 이내 | `3차 매수선 1% 인접` | 0% < 이격도 ≤ 1% |
| **BOUGHT_2** | 저가 기준 3% 이내 | `3차 매수선 3% 인접` | 1% < 이격도 ≤ 3% |
| **BOUGHT_2** | 저가 기준 5% 이내 | `3차 매수선 5% 인접` | 3% < 이격도 ≤ 5% |

**중복 알람 방지**
- 동일 종목, 동일 조건당 **하루 1회**만 전송
- `alert_history.json` 파일에 기록

#### 전송 내용

**매수선 접근 알람**
- 알람 타입 (예: "1차 매수선 5% 인접")
- 현재 시각 (HH:MM:SS)
- 종목명
- 현재가
- 목표가 (매수선)
- 이격도 (%)

**매수 체결 알람** (추가 정보 포함)
- 위 정보 +
- 3% 매도가
- 5% 매도가
- 7% 매도가

#### 메시지 포맷 예시

**매수선 접근:**
```
🟡 1차 매수선 5% 인접
🕐 14:25:30
───────────
종목: 삼성전자
현재가: 77,800원
목표가: 75,500원
이격도: +3.05%
```

**매수 체결:**
```
✅ 1차 매수 체결!
🕐 14:30:15
───────────
종목: 삼성전자
현재가: 75,300원
목표가: 75,500원
이격도: -0.26%

3% 매도가: 77,665원
5% 매도가: 79,445원
7% 매도가: 81,225원
───────────
```

#### 전송 대상
- **수신자**: 모든 사람 (`recipients=["all"]`)
- **전송 시점**: 조건 충족 즉시

## 🔑 핵심 차이점

### 1. **이격도 계산 기준**
- **일일 리포트**: `현재가` vs `익일 기준 매수선`
- **실시간 알람**: `저가` vs `당일 기준 매수선`

### 2. **알람 임계값**
- **일일 리포트**: 10% 이내 (기본값, `--alert-threshold` 옵션)
- **실시간 알람**: 1%, 3%, 5% 단계별

### 3. **매수선 기준**
- **일일 리포트**: 익일 기준 매수선 (예측값)
- **실시간 알람**: 당일 기준 매수선 (실제 체결 판단용)
