## 3. 일일 트레이딩 리포트

### 함수
- `send_slack_daily_report()` (slack_notifier.py 309줄)
- **현재 상태**: Block Kit 형식이 아닌 일반 텍스트 형식

### 예시 데이터
```python
{
    "alerts": [
        {
            "종목명": "삼성전자",
            "종가": 107700,
            "1차매수선(익일)": 75500,
            "1차매수선이격도(%)": -29.9,
            "알람상태": "READY_BUY1"
        },
        {
            "종목명": "SK하이닉스",
            "종가": 560000,
            "평균매수가": 138000,
            "매수상태": "BOUGHT_1",
            "알람상태": "BOUGHT"
        }
    ],
    "total_stocks": 79,
    "system_label": "S2"
}
```

### 현재 텍스트 형식 (Block Kit 아님)
```
📊 *[S2] 일일 트레이딩 리포트*
🕐 2025-11-02 20:10
────────────

🟡 *1차 매수 접근 중* (2개)
  • 삼성전자
    현재가: 107,700원 / 매수가: 75,500원 / 이격도: -29.9%

🔴 *매수 완료 종목* (1개)
  • SK하이닉스
    현재가: 560,000원 / 평균가: 138,000원 / 이격도: +305.8%

🟢 *매도선 접근* (1개)
  • NAVER
    현재가: 270,500원 / 목표가: 225,570원 / 이격도: +19.9%
```

### 현재 Block Kit JSON
**없음** - 현재는 일반 텍스트 형식만 사용

## 🔧 수정 필요 사항

### 1. 매수선 인접 알람
- 현재 양식 확인 필요
- 레이아웃 개선 가능 여부 확인

### 2. 매수 체결 알람
- 현재 양식 확인 필요
- 매도가 정보 표시 방식 개선 가능 여부 확인

### 3. 일일 트레이딩 리포트
- **Block Kit 형식으로 전환 필요**
- 현재 텍스트 형식 → Block Kit 형식 변환
- 섹션별 그룹화 및 정렬 개선

---

## 📝 참고사항

- 모든 가격 포맷: 천 단위 콤마 (예: 77,800원)
- 이격도 포맷: 소수점 2자리 (예: +3.05%, -0.26%)
- 시간 포맷: 실시간 알람은 `HH:MM:SS`, 일일 리포트는 `YYYY-MM-DD HH:MM`
- 시스템 라벨: `S1` 또는 `S2` (자동 감지)
