# 🎯 자동 매매 시그널 시스템 완전 가이드

## 📋 시스템 개요

**20일 이평선 엔벨로프 -20% 지지선 기반 3단계 분할 매수/매도 자동 시그널 시스템**

### 핵심 전략
```
[20일 이동평균선]
    ↓ -20% (엔벨로프)
[지지선 + 1호가] → 1차 매수선 (A)
    ↓ -10%
[2차 매수선 (B)]
    ↓ -10%
[3차 매수선 (C)]

매도:
- 평균매수가 +3% 도달 → 하락 → +3% 재터치 = 전량매도
- 평균매수가 +5% 도달 → 하락 → +5% 재터치 = 전량매도
- 평균매수가 +7% 도달 = 즉시 전량매도
```

## 🚀 사용 방법

### 1단계: 유니버스 생성 (5천억+ 종목 수집)


**결과:** `turnover_universe.xlsx` 생성 (종목 목록)

### 2단계: 시그널 생성 (매매 시그널 분석)

```bash
# 기본 실행 (알람 임계값 10%)
python Trading_Signal_System.py

# 알람 임계값 변경 (5%로 민감하게)
python Trading_Signal_System.py --alert-threshold 5

# 유니버스/시그널 파일 지정
python Trading_Signal_System.py --universe my_universe.xlsx --signal my_signals.xlsx
```

## 📈 이격도 활용 방법

### 예시 1: 삼성전자 (매수 전)
```
종가: 62,500원
1차매수선: 48,646원
1차매수선이격도: +28.5%

→ 해석: 28.5% 더 떨어져야 1차 매수
→ 액션: 관찰만 (알람 없음)
```

### 예시 2: SK하이닉스 (1차 매수 후)
```
종가: 52,000원
평균매수가: 55,000원
2차매수선: 49,500원
2차매수선이격도: +5.1%

1차매도선: 56,650원 (+3%)
1차매도선이격도: -8.2%

→ 해석: 
  - 5.1% 더 떨어지면 2차 매수 (접근 중!)
  - 8.2% 올라야 +3% 매도선
→ 액션: 2차 매수 준비 (🔴🔴 알람)
```

### 예시 3: NAVER (3차 매수 완료)
```
종가: 208,000원
평균매수가: 195,000원
3차매도선: 208,650원 (+7%)
3차매도선이격도: -0.3%

→ 해석: 0.3%만 더 오르면 +7% 도달!
→ 액션: 전량 매도 준비 (💰💰💰 알람)
```

## 🎯 매수/매도 체결 판단

### 매수 체결 로직
```python
if 당일저가 <= 매수선:
    매수 체결됨!
```

**예시:**
- 1차매수선: 48,646원
- 당일저가: 48,500원
- → 1차 매수 체결! ✅

### 매도 재터치 로직
```python
최고도달선 기록:
if 당일고가 >= +3%:
    최고도달선 = +3%
    
재터치 감지:
if 최고도달선 >= +3% and 종가 <= +3%:
    +3% 재터치! 전량 매도!
```

**예시:**
- Day 1: 고가가 +3.5% 도달 → 최고도달선 = +3.5%
- Day 2: 고가 +2.8% → 변화 없음
- Day 3: 고가 +3.1% 재터치 → 전량 매도! 💰

## 📝 사용 워크플로우

### 초기 설정
1. 유니버스 생성: `python Daily_Turnover_Tracker.py`
2. 시그널 생성: `python Trading_Signal_System.py`
3. `trading_signals.xlsx` 열어서 확인

### 매일 운영
1. **장 시작 전 (8시 10분)**
   - 유니버스 업데이트 (전일 거래대금 상위)
   
2. **장 마감 후 (20시 10분)**
   - 시그널 분석 실행
   - 알람 확인 (콘솔 출력 또는 텔레그램)
   
3. **알람 발생 시**
   - `READY_BUY1/2/3`: 매수 준비
   - `READY_SELL1/2/3`: 매도 준비
   - 엑셀 파일에서 이격도 확인
   
4. **매수 체결 후**
   - 엑셀에서 수동으로 **매수량** 입력 (금액이 각자 다르므로)
   - 다음 실행 시 자동으로 평균매수가, 매도선 계산됨

### 매수 후 수량 입력 방법
```
trading_signals.xlsx → Summary 탭

1차 매수 체결 후:
- 1차매수량 셀에 직접 입력 (예: 100주)
→ 다음 실행 시 자동으로:
  - 평균매수가 계산
  - 1차/2차/3차 매도선 계산
  - 매도선 이격도 표시

2차 매수 체결 후:
- 2차매수량 셀에 직접 입력 (예: 150주)
→ 평균매수가 재계산

3차 매수 체결 후:
- 3차매수량 셀에 직접 입력 (예: 200주)
→ 최종 평균매수가 확정
```

## 💡 팁

1. **알람 임계값은 5~10% 추천**
   - 너무 낮으면 너무 많은 알람
   - 너무 높으면 타이밍 놓침

2. **매수 비율은 1:1:1 권장**
   - 각 단계에 동일 금액 투자
   - 리스크 분산

3. **엑셀 파일은 매일 백업 권장**
   - History 탭에 소중한 매매 기록

4. **누적횟수가 높은 종목 주목**
   - 15회 이상 = 매우 강한 종목
   - 지속적 거래대금 상위권 유지

5. **이격도 해석**
   - 양수(+): 더 떨어져야 함 (매수선) 또는 더 올라야 함 (매도선)
   - 음수(-): 이미 도달 (매수 체결) 또는 초과 (매도 가능)
   - 절대값이 작을수록 타겟에 가까움
